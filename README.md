# Demo Maps

A simple React map application using Leaflet and OpenStreetMap.

## Dependencies

- React 19 + TypeScript
- Vite
- Leaflet / React-Leaflet
- Tailwind CSS v4
- [Node Version Manager](https://github.com/nvm-sh/nvm)

## Setup

1.1. (Optional) If using NVM

```bash
nvm install
nvm use
```

1.2 Install dependencies:

```bash
npm install
```

2. Create environment file:

```bash
cp .env.example .env
```

3. Start development server:

```bash
npm run dev
```

## Scripts

| Command | Description |

| `npm run dev` | Start dev server |
